Rhythm Chart Generator (데스크톱 버전)
======================================

■ 실행 방법
  1. RhythmChartGenerator.exe 를 더블클릭합니다.
  2. 검은 콘솔 창이 뜨고 잠시 후(첫 실행은 10~20초) 기본 브라우저가
     자동으로 열립니다. (열리지 않으면 콘솔에 표시된 http://127.0.0.1:####
     주소를 직접 브라우저에 붙여넣으세요.)
  3. 브라우저 화면에서 WAV 파일 업로드 또는 유튜브 링크로 채보를 만듭니다.
  4. 종료하려면 검은 콘솔 창을 닫으면 됩니다.

■ 특징
  - 별도 설치 필요 없음. Python, ffmpeg, Node 없이 이 exe 하나로 동작합니다.
    (ffmpeg는 실행파일 안에 포함되어 있습니다.)
  - 만들어진 채보는 브라우저의 "내보내기" 버튼으로 저장됩니다.

■ 데이터 폴더
  - 실행하면 exe 옆에 data\downloads 폴더가 생겨 임시 오디오가 저장됩니다.
    지워도 무방하며, 새 곡을 넣으면 이전 임시 파일은 자동 정리됩니다.

■ 주의
  - Windows SmartScreen 경고가 뜨면 "추가 정보" → "실행"을 누르세요.
    (서명되지 않은 실행파일이라 뜨는 정상적인 경고입니다.)
  - 유튜브 링크 기능은 인터넷 연결이 필요합니다.
